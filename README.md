# Cryptocurrency App

**Status:** Complete ✅

**Completion Status:** 100% ✅

## Overview

- This app displays the ranks of all the public cryptocurrencies in existence.
- It uses Jetpack Compose and MVVM Clean Architecture.
- It fetches data from a third-party REST-API
